WPC DAQ Device Programming Guide
===================================


Overview
--------
WPC Python-based driver contains an API for interacting with the WPC USB-DAQ, Ethernet and Wifi-DAQ series products. 

Also, It has been tested under Window 10 operating systems. For more products information, please visit `WPC-Systems-Ltd <http://www.wpc.com.tw/>`_.

Examples
--------
For WPC DAQ Python driver console or GUI examples, please go check `GitHub Examples <https://github.com/WPC-Systems-Ltd/WPC_Python_driver_release/tree/main/Examples>`_.

Driver Version
--------------
The latest released version of WPC DAQ Python driver is 0.2.3 from September 1, 2022.

For the latest development, archived versions and code repository, please visit `GitHub source code <https://github.com/WPC-Systems-Ltd/WPC_Python_driver_release>`_.
