class pywpc.USBDAQF1CD

This device WPC-USB-DAQ-CD has 20ch 3.3V DIO/I2C/SPI

  • UART : port1, port2

  • I2C : port1, port2

  • SPI : port2

  • CAN : port1

  • DIO : port0, port1, port2, port3
