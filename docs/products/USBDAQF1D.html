Write digital value to specefic pin.

Parameters

  • Port index from WPC Device.

pinint or list

  • Pin number you want to set.

  • For example, you want to open pin 0, pin 1 and pin 5.

  • pin = [0,1,5].

  • Pin number you want to set high or low.

async sys_getRTC()

Get date and time to connected WPC device.

Returns
str_str

Date and time represented by “A-B-C, D:E:F”, where

  • “A” is year

  • “B” is month

  • “C” is day

  • “D” is hour

  • “E” is minute

  • “F” is second
