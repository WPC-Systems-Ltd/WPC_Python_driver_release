Device pinout

API function description

class pywpc.EthanP

  1. Power input: 24VDC

  2. 10/100 T-based Ethernet interface

  3. 4ch variable digital resistor output

  4. Output range: 300Ω to 50kΩ

  5. Max. update rate: 500Hz
