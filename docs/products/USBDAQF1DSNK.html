class pywpc.USBDAQF1DSNK

This device WPC-USB-DAQ-DSNK has 24V 12ch DO & 14ch DI.

Available ports:

  • DI: 0, 1

  • DO: 2, 3

async DO_writePins(port, pin, value)

Write digital value to specific pin.

Parameters
portint

Available port number depends on WPC DAQ device.

pinint or list

  • Pin number you want to set.

  • For example, you want to open pin 0, pin 1 and pin 5.

  • pin = [0,1,5].

valueint or list

  • Pin number you want to set high or low.

  • High = 1, Low = 0.

  • For example, set pin 0 and pin 1 to high, and pin 5 to low.

  • value = [1,1,0].

Returns
statusint

0 if success, error otherwise.

async Sys_getRTC()

Get date and time to connected WPC device.

Returns
str_str
Date and time represented by “A-B-C, D:E:F”.

  • “A” is year.

  • “B” is month.

  • “C” is day.

  • “D” is hour.

  • “E” is minute.

  • “F” is second.
