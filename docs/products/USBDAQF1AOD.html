USB-DAQ-F1-AOD

API function description

class pywpc.USBDAQF1AOD

This device WPC-USB-DAQ-AOD has 16ch 3.3V DIO/I2C/SPI & 8ch AI & 8ch AO.

Available ports:

  • AI: 0

  • AO: 0

  • UART: 1, 2

  • I2C: 1, 2

  • DIO: 0, 1, 2, 3
