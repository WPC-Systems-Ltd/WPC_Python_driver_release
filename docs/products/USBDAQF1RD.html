USB-DAQ-F1-RD

Device pinout

API function description

class pywpc.USBDAQF1RD

This device WPC-USB-DAQ-RD has 2ch thermocouple input which support PT-100 or PT-1000 and 21ch 3.3V DIO/I2C/SPI.

Available ports:

  • UART: 1, 2

  • I2C: 1, 2

  • SPI: 2

  • RTD: 1

  • DIO: 0, 1, 2, 3
